# UE Asset Prep & Validation Tool

Editor Utility tool for Unreal Engine 5.x that validates and auto-fixes production assets.

## Features (Planned)
- Asset naming validation
- Texture & mesh checks
- Batch folder scanning
- Auto-fix with undo support
- JSON reporting

## Tech
- Unreal Engine 5.x
- Editor Utility Widgets
- Unreal Python
